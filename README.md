# Job Application Tracking System

A CLI-based application built using Python and MySQL to track and manage job applications efficiently.

## Features
- Add new job applications
- View all applications
- Update application status
- Delete applications
- Parameterized queries (SQL Injection safe)
- Exception handling
- OOP-based architecture
- MySQL database integration

## Technologies Used
- Python
- MySQL
- mysql-connector-python
- Git & GitHub

## Database Schema

Table: applications

- id (Primary Key, AUTO_INCREMENT)
- company_name (VARCHAR)
- role (VARCHAR)
- location (VARCHAR)
- salary (DECIMAL)
- status (VARCHAR)
- applied_date (DATE)
- notes (TEXT)
